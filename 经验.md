# 经验

- pcduino 带的 3.3V 输出足够舵机的脉冲控制了，不需要升压的电平板

- 输出脉冲的时候，注意一定要接 pcduino 的地线，不然没回路的

- 可以用 analog 去输 pwm 信号

- analog 的 maxValue 是 32

- 用 analog 可能是舵机转不到 180，因为损失了 1/32 的精度
